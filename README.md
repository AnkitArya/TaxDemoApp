# TaxApp

## Dependencies
1. .Net Core 2.1.300
2. NodeJs
3. VS-Code

## Steps to Run
1. dotnet restore 
2. Run "npm i" in command line in pacjage.json file directory
2. F5

## Steps for Client Test Cases:
1. Move to package.json Directory
2. Run "npm test"

## Steps for Server Test Cases:
1. Move to Test Directory
2. Run "dotnet test"
